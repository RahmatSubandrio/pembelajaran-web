Cerita Saya

Nama saya Subandrio, seorang mahasiswa jurusan Teknologi Informasi yang penuh semangat dan dedikasi. Perjalanan saya dalam dunia pemrograman web dimulai dari ketertarikan saya terhadap teknologi dan bagaimana internet telah mengubah cara kita berinteraksi dengan dunia. Sejak awal kuliah, saya merasa tertantang untuk memahami dasar-dasar pemrograman web dan mengembangkan keterampilan yang diperlukan.

Saat memulai perjalanan ini, saya terlebih dahulu mengumpulkan pengetahuan dasar tentang HTML, CSS, dan JavaScript. HTML sebagai tulang punggung struktur halaman web, CSS untuk menghias tampilan halaman, dan JavaScript sebagai bahasa pemrograman yang memberikan interaktivitas pada situs web. Awalnya, saya menghadapi beberapa kesulitan dalam memahami konsep-konsep ini, tetapi dengan tekad yang kuat, saya terus berlatih dan mencari sumber belajar yang tepat.

Sumber belajar daring seperti tutorial, forum, dan platform pembelajaran seperti Codecademy, freeCodeCamp, dan Coursera menjadi sahabat sehari-hari saya. Saya merancang waktu untuk belajar di antara jadwal kuliah dan tugas lainnya. Prosesnya tidak selalu mulus, namun setiap kali saya berhasil menyelesaikan suatu proyek atau memahami konsep baru, rasa puas itu sangat memotivasi saya untuk terus maju. Seiring berjalannya waktu, kemampuan saya dalam mengembangkan proyek-proyek kecil semakin meningkat. Saya mulai membangun halaman web sederhana untuk menampilkan informasi tentang diri saya dan proyek-proyek yang telah saya selesaikan. Setiap proyek itu merupakan langkah maju, membantu saya memahami bagaimana menggabungkan HTML, CSS, dan JavaScript secara sinergis.
